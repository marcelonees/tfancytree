# TFancytree - Componente para Adianti Framework

Componente que integra a biblioteca Fancytree ao Adianti Framework, oferecendo uma árvore hierárquica com recursos avançados.

## Características

- ✅ Checkboxes para seleção múltipla
- ✅ Drag & Drop
- ✅ Edição inline de nós
- ✅ Lazy loading (carregamento sob demanda)
- ✅ Filtro/pesquisa
- ✅ Temas customizáveis
- ✅ Eventos completos (click, select, expand, etc.)

## Instalação

```bash
composer config repositories.tfancytree vcs https://github.com/marcelonees/tfancytree
composer require marcelonees/tfancytree:dev-main
